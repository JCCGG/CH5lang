# 特性

产物大小极小，按需编译，无任何内置运行时植入，完全可控。

编译速度快，原生支持调用系统api。

目前支持的平台有：x86

# 使用方法

x86平台编译方法：

`java -jar ch5.jar -c 1.ch5 1.exe`

将`1.ch5`源代码文件编译成32位可执行文件`1.exe`
