# 语法介绍

语法的解析是从关键字开始的，关键字分为2大类：
1语法
2语法

1语法为外层关键字，有：
```
import
class
static
fun
var
val
```

2语法是函数内关键字，有：
```
if
for
var
val
break
continue
fun
```

其中部分关键字重复说明函数内外都可以使用该关键字
